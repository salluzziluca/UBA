Para probar el cero por acotado aplicamos el siguiente paso a paso
1. $\frac {(x+2)^2 * y²}{(x+2)^2 + y²}$ 
2. $y²=y²$
3. y²<=$y²+(x+2)^2$
4. 0<=$\frac { y²}{(x+2)^2 + y²}$ <= 1 está acotado y $(x+2)²$ tiende  a cero
